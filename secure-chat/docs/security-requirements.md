# Security Requirements

REQ-001:
Messages shall be encrypted end-to-end.

REQ-002:
Server shall not possess session keys.

REQ-003:
Replay attacks shall be detected.

REQ-004:
All packets shall be authenticated.

REQ-005:
Passwords shall be protected using Argon2id.

REQ-006:
Session keys shall be derived using ECDH.

REQ-007:
Local chat history shall be encrypted.
