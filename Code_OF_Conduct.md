# Code of Conduct

## Our Commitment

We are committed to fostering an open, professional, and respectful environment for all contributors regardless of:

* experience level
* nationality
* gender identity
* religion
* disability
* background

The project prioritizes ethical collaboration, responsible security research, and constructive communication.

---

# Expected Behavior

Contributors are expected to:

* Use respectful and professional language
* Accept constructive criticism
* Focus on technical collaboration
* Report vulnerabilities responsibly
* Respect privacy and security principles
* Support inclusive participation

---

# Unacceptable Behavior

The following behaviors are prohibited:

* Harassment or discrimination
* Personal attacks
* Malicious code contributions
* Intentional security sabotage
* Publishing private information
* Spam or disruptive conduct

---

# Responsible Security Research

Because this project involves cryptography and secure communications:

* vulnerabilities must be disclosed responsibly
* exploit demonstrations must remain ethical
* contributors must avoid harmful deployment

---

# Enforcement

Project maintainers reserve the right to:

* reject contributions
* remove harmful content
* suspend participation
* ban malicious contributors

when necessary to protect the integrity and safety of the project.

---

# Reporting Issues

Security or conduct concerns should be reported privately to the maintainers before public disclosure.
